# Font Files

Please place your **MiguerSans-Regular** font files here:

- `MiguerSans-Regular.ttf` (TrueType Font)
- `MiguerSans-Regular.woff2` (Web Font Format 2 - recommended)
- `MiguerSans-Regular.woff` (Web Font Format - fallback)

## Instructions:

1. Copy your MiguerSans-Regular font files to this directory
2. Make sure the filenames match exactly: `MiguerSans-Regular.ttf`, `MiguerSans-Regular.woff2`, `MiguerSans-Regular.woff`
3. The font will be automatically loaded and used for MVP headings (your name and main title)

## Font Usage:
- **MVP Headings**: Uses MiguerSans-Regular (font-display class)
- **Section Titles**: Uses Space Grotesk (font-heading class)
- **Body Text**: Uses Inter (font-sans class)
